## Description

My Fisrt NestJS API.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

##  docker

```bash
# running docker 
$ docker compose up
```

## Prisma

```bash
# creaing prisma migrates
$ npx prisma migrate dev

# running prisma studio
$ npx prisma studio
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
